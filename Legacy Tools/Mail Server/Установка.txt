Вариант 1. MailCow Dockerized
Проверить нет ли занятых портов:
ss -tlpn | grep -E -w '25|80|110|143|443|465|587|993|995|4190'
Установить Docker&Docker Compose
Проверить командой umask параметр - должен быть 0022
cd /opt
git clone https://github.com/mailcow/mailcow-dockerized
cd mailcow-dockerized
./generate_config.sh - после запуска , предложит создать имя домена , создаст ключ и время региона и на выбор версию сервера (выбираем стабильную)
nano mailcow.conf - основной конфиг , там можно править все нужные конфиги , например http на 8080 или https на 8443
docker compose pull - запустит установку всех модулей для Mailcow 
docker compose up -d - запустит контейнер после окончания установки (будут созданы так же volumes для контейнеров)
После ввести в браузере адрес созданного домена (логин:admin пароль:moohoo)
Сразу сменить данные в админке логин и пароль.Зайти в настройки почты и создать домен name.net , установить длину DKIM ключа на 1024 , т.к 2048 не все сервисы поддерживают.
