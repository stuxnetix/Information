sudo chown name* -R /root/ - сделать текущего пользователя рутом (при условии , что может использовать команду sudo)
netstat -tulpen -> просмотреть IP и порты которые открыты с машины
ip a - ip машины , подробная инфа
env - все переменные в системе
export NAME_TEST=test -> создать свою переменную для env
pwd - показать текущую директорию
mkdir - создать папку
rmdir - удалить папку
rm Name.txt* - удалить файл
mv Dir1 Dir2 - переместить папку 1 в папку 2 (можно указывать путь /home/root/Desktop/Name2)
find - поиск файлов
find /etc -name hosts - выдаст в этой папке папки hosts , если прописать "hosts" , выдаст все файлы и папки с этим именем и расширениями к нему.
ps aux - показать все процессы , которые запущены в системе
kill -signal PID - убить процесс с IDname
term -signal PID - завершить процесс с IDname
chgrp - файл группы изменяет владельца группы
chmod - файл прав изменяет права доступа к файлу (chmod u=rwx,g+rw,o-r , дает полный доступ к файлу , права на группу запись и чтение , остальным только чтение)
Буква a=rx (дает всем 3 категориям равные права)
chmod 754 - 7 чтение , 5 - запись , 4 - исполнение ( математика 4+2+1=7)
chmod -R a+x DIR , даст права на выполнение всем , в текущей папки и все что в ней
free - команда о памяти 
disk free -h -m (mbytes) or -g(gbytes) (df) - инфа о памяти на каждом диске
id - личность юзера , его группа 
uname -a - инфа о ОС 
systemd - журнал 
journalctl -u name.service - инфо о процессе , параметр -r все новые записи
pcutils - все PCI - устройства
usbutils - все usb устройства
pcmciautils - все PCMCIA карты (обычно требует установки через apt install)
bg - отправить процесс на передний план
adduser - добавить user'a
getent - команда для обращения к БД пользователя (getent passwd USER) 
passwd - сменить пароль (обновляет файл shadow)
chfn - сменить имя пользователя
chsh - сменить оболочку входя пользователя
chage- - изменить срок действия пароля , указать имя пользователя -I 
passwd -e USER - при следующем входе в систему , попросит сменить пароль
passwd -l USER - залочить пользователя , -u разблокировать
addgroup - добавить группу 
delgroup - удалить группу
groupmod - изменяет информацию о группе ее gid и идентификатор
gpasswdgroup - изменяет пароль группы 
gpasswd -r GROUP - удаляет пароль группы
systemctt start ssh - запустить службу ssh (/etc/ssh/sshd_config)
systemctl status - список всех служб
